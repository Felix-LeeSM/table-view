## F3.1: 데이터 그리드 뷰 확장

- [x] 페이지네이션 지원 (기본 300 rows/page, 설정 가능)
- [x] 페이지 크기 선택기 제공 (100, 300, 500, 1000)
- [x] 첫 페이지 / 마지막 페이지 이동 버튼
- [x] 특정 페이지 번호로 직접 이동 (Jump to page)
- [x] 컬럼 순서 변경 가능 (드래그앤드롭)
- [x] BLOB/Binary 데이터는 아이콘으로 표시, 클릭 시 상세 보기

## F3.2: 인라인 편집

- [x] 셀 더블클릭 시 편집 모드 진입
- [x] 텍스트, 숫자, boolean, 날짜/시간 타입 편집 가능
- [x] 편집 중인 셀이 시각적으로 구분됨 (노란색 배경)
- [x] Tab/Enter로 다음 셀로 이동
- [x] Escape로 편집 취소
- [x] 변경 사항은 즉시 DB에 반영되지 않고 "pending" 상태로 표시
- [x] "Commit" 버튼(Cmd+S)으로 변경 일괄 실행
- [x] "Discard" 버튼으로 모든 보류 변경 취소
- [x] Commit 전 생성되는 SQL 미리보기 (Code Review)

## F3.3: 행(Row) 조작

- [x] "Add Row" 버튼으로 빈 행 삽입
- [x] 행 선택 후 "Delete Row"로 삭제 (pending 상태)
- [x] "Duplicate Row"로 선택한 행 복제 삽입
- [x] 여러 행 동시 선택 가능 (Shift+Click, Cmd+Click)
- [x] 선택한 행 복사 (Cmd+C): Plain text, JSON, CSV, SQL Insert 형식
- [x] 우클릭 컨텍스트 메뉴: Edit, Delete, Duplicate, Copy as...

## F4.2: 쿼리 실행 UI 확장

- [x] 쿼리 에디터 상단 툴바에 실행(Run) 버튼 표시 (▶ 아이콘)
- [x] 쿼리 실행 중 툴바에 취소(Cancel) 버튼으로 전환 (■ 아이콘)
- [x] Cmd+Return / Ctrl+Return 단축키 안내 표시

## F4.3: 미리보기 탭 확장

- [x] 테이블 클릭 시 "미리보기" 탭으로 열림 (임시)
- [x] 다른 항목 탐색 시 기존 미리보기 탭이 교체됨
- [x] 편집, 페이지 이동, 또는 명시적 핀 클릭 시 영구 탭으로 승격
- [x] 미리보기 탭은 이탤릭체 제목으로 영구 탭과 시각적 구분
- [x] 탭에 연결 정보(색상) 표시

## F4.5: 쿼리 포맷팅

- [x] Cmd+I: SQL Beautify (들여쓰기, 줄바꿈으로 정렬)
- [x] Cmd+Shift+I: SQL Uglify (한 줄로 압축)
- [x] 포맷팅이 선택 영역에만 적용 가능

## F4.6: 쿼리 히스토리

- [x] 모든 실행 쿼리가 자동으로 히스토리에 기록됨
- [x] 히스토리 패널에서 과거 쿼리 검색 가능
- [x] 과거 쿼리 클릭 시 에디터에 삽입
- [x] 연결별 히스토리 분리
- [x] 히스토리 항목에 실행 시간, 타임스탬프 표시

- [x] 페이지 크기 선택 및 페이지 이동 동작
- [x] 사이드바 인터랙티브 요소 텍스트 선택 방지
