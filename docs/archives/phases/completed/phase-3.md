# Phase 3: Query Editor

> 쿼리 작성 및 실행 환경 — **완료**

## 구현 항목

| Feature | ID | 우선순위 |
|---------|-----|---------|
| SQL 에디터 기본 (F4.1) | F4.1 | P0 |
| 쿼리 실행 및 결과 (F4.2) | F4.2 | P0 |
| 다중 탭 (F4.3) | F4.3 | P0 |

## F4.1: SQL 에디터 기본

- [x] SQL 입력 영역이 monospace 폰트로 표시됨
- [x] 구문 하이라이팅: 키워드(파랑), 문자열(초록), 숫자(주황), 주석(회색)
- [x] 줄 번호 표시
- [x] 들여쓰기 자동 적용
- [x] 괄호 자동 닫기 및 매칭 하이라이트
- [x] Cmd+Return으로 현재 쿼리(또는 커서 위치 쿼리) 실행
- [ ] Cmd+Shift+Return으로 전체 쿼리 실행
- [x] 세미콜론(;)으로 여러 쿼리 구분 가능

## F4.2: 쿼리 실행 및 결과

- [x] 실행된 쿼리의 결과가 하단 그리드에 표시됨
- [x] 결과 그리드는 테이블 브라우저(FA-3)와 동일한 UX
- [x] 실행 시간 표시 (예: "245ms, 1,234 rows")
- [x] 쿼리 실행 중 로딩 스피너 표시
- [x] 장시간 실행 쿼리 취소 가능 (Cancel 버튼 또는 Cmd+.)
- [x] 에러 발생 시 빨간색 에러 메시지 표시 (라인 번호 포함)
- [x] DML(INSERT/UPDATE/DELETE) 실행 시 "N rows affected" 표시
- [x] DDL 실행 시 "Query executed successfully" 표시
- [x] 쿼리 에디터 상단 툴바에 실행(Run) 버튼 표시 (▶ 아이콘)
- [x] 쿼리 실행 중 툴바에 취소(Cancel) 버튼으로 전환 (■ 아이콘)
- [x] Cmd+Return (macOS) / Ctrl+Return (Windows/Linux) 단축키 안내 표시

## F4.3: 다중 탭

- [x] 탭 바에 열린 탭 목록 표시
- [x] 새 탭 생성 버튼(Cmd+T)
- [x] 탭 닫기(X 버튼, Cmd+W, 휠 클릭)
- [x] 탭 간 전환 시 각 탭의 쿼리 내용과 결과 유지
- [x] 탭에 연결 정보(색상) 표시
- [x] 서로 다른 연결의 탭을 동시에 열어둘 수 있음
- [x] 앱 재시작 시 이전 탭 상태 복원
- [x] 테이블 클릭 시 "미리보기" 탭으로 열림 (임시)
- [x] 다른 항목 탐색 시 기존 미리보기 탭이 교체됨
- [x] 편집, 페이지 이동, 또는 명시적 핀 클릭 시 영구 탭으로 승격
- [x] 미리보기 탭은 이탤릭체 제목으로 영구 탭과 시각적 구분

## F4.4: 자동완성

- [x] 테이블 이름 자동완성 (스키마 접두사 포함)
- [x] SQL 키워드 자동완성 (SELECT, FROM, WHERE, JOIN...)
- [x] Tab/Enter로 자동완성 선택
- [x] ↑↓ 화살표로 자동완성 목록 탐색
- [x] ESC로 자동완성 목록 닫기
- [x] 컬럼 이름 자동완성 (테이블 컨텍스트 기반)
- [x] 함수 이름 자동완성
- [ ] 자동완성 목록에 타입/설명 부가 정보 표시
