Production/Staging Warning Banner — 4 Variants

현재 baseline (4px 색띠) → 텍스트가 있는 색띤 메시지로 교체. 5 환경 → production(빨강) / staging(주황) / 나머지 3개(무표시) 로 정리. 아래 4개 variant 를 비교해서 위치/스타일 결정.

Baseline (제거 대상): 현재는 dialog 본문 최상단에 4px 높이의 색띠 1줄만 (production = #ef4444). 텍스트 없음.

A) Body 최상단 배너 (추천)

PreviewDialog 의 `error` / `commitError` 와 같은 패턴 — body 안에 배너로. Execute 버튼 한 번 클릭으로 끝. 색띠 제거.
