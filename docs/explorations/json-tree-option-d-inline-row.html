Option D — Inline row expand (cell 안 토글 버튼)

Expand 트리거는 nested cell 자체. OBJ → { ... } 의 … 가 버튼, ARR → [ n items ] 의 n items 가 버튼. 한 번 누르면 열리고 (cell 의 토글이 변형 { ✕ }), 같은 위치 다시 누르면 닫힘. detail row 우상단에도 동일하게 닫는 ✕. 다른 nested 셀 클릭하면 같은 detail 영역이 그 셀의 path 로 교체.
