**결정**: Sidebar의 2-letter 좌측 레일을 제거하고 "Connections / Schemas" 두 모드 탭으로 분리. 상단 토글로 전환, Schemas 모드는 connection 미선택 시 Connect CTA만 표시.
**이유**: 2-letter 레일은 연결이 많아지면 식별 불가. 모드별로 좌측 리스트 내용을 완전히 교체해 "연결 관리"와 "스키마 탐색" 두 관심사를 시각적으로 분리.
**트레이드오프**: + 정보 밀도 감소, 신규 유저 학습 용이 / - 두 모드 간 왕복 클릭 비용, 빈 Schemas 모드 처리 로직(Connect CTA) 추가 필요.
