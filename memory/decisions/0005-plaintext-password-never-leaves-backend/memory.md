---
id: 0005
title: plaintext 비밀번호는 IPC 경계를 넘지 않는다
status: Accepted
date: 2026-04-20
---

**결정**: `ConnectionConfig`의 `password` 필드는 백엔드에서 프론트로 직렬화할 때 항상 빈 문자열로 마스킹. 프론트는 연결 편집 시에만 사용자 입력으로 받아 백엔드에 전달하며, 저장된 비밀번호를 UI로 복원하는 경로는 없음.
**이유**: AES-256-GCM 저장만으로는 부족 — 프론트 메모리/DevTools/로그/스크린샷에 노출되는 순간 로컬 악성 확장으로 유출 가능. IPC 경계를 최소 권한 경계로 취급해 "프론트는 절대 plaintext를 모른다"를 불변식으로.
**트레이드오프**: + 프론트 side-channel 전부 차단 / - "비밀번호 보기" UI 불가, export 결과도 마스킹되어 import 시 사용자가 재입력해야 함.
